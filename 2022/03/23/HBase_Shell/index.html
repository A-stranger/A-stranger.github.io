数据操作语言

数据操作语言（Data Manipulation Language, DML），用户可以使用它操纵数据，实现对数据库的基本操作，如查询、插入、删除、修改等。

编号为 0001， info { name=zhangsan }

设置学生0001基本信息info { age=18 sex=Male }

设置学生0001成绩score { hadoop=30 hbase=35 } VERSION=1

以上，完成了一条学生记录的数据插入。
