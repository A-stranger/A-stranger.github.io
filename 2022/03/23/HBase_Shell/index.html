过滤器

在HBase中，get和scan都可以使用过滤器(Filter)来设置查询范围。类似于SQL里面的where条件查询。

过滤器语法

scan/get ‘表名’，{FILTER=> “过滤器 ( 比较运算符，’比较器’)”}

过滤器
比较器
比较器 描述
BinaryComparator 匹配完整字节数组
BinaryPrefixComparator 匹配字节数组前缀
BitComparator 匹配比特位
NullComparator 匹配空值
RegexStringComparator 匹配正则表达式
SubstringComparator 匹配子字符串
比较运算符
比较运算符 描述
= 等于
> 大于
>= 大于等于
< 小于
<= 小于等于
!= 不等于

过滤器分类

  • 列族与列过滤器

  • 值过滤器

  • 其他过滤器
