引入方式

内联样式

内联样式也称为行内样式表，CSS是在元素的开始标签的style属性中定义的。

内部样式

内部样式（internal）在<head>区域中的<style>区域中定义

外部引用

外部样式表（external）在单独文件中定义，然后在HTML文件的<head></head>标签对中使用link标签来引用。

CSS

在实际开发中，一般都是使用外部样式表比较多，但是并不代表内部样式表以及行内样式表就一无是处。
